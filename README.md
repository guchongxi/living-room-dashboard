# Living Room Dashboard

客厅大屏页面 — 时钟 + 天气显示。

## 预览

![screenshot](screenshot-v3.png)

直接打开 `index.html` 即可在浏览器中查看。

## 功能

- 实时时钟显示
- 天气信息展示
- 支持 iOS 10.3.3 添加到主屏幕（standalone 模式）
- 支持 PWA manifest（iOS 16.4+ / Android Chrome）

## 使用

```bash
# 本地查看
open index.html
# 或用任意静态服务
python3 -m http.server 8080
```
